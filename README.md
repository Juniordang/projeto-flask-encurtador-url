# Encurtador de URLs com Flask

Projeto desenvolvido utilizando Python e o micro-framework Flask para transformar URLs longas em links curtos.

## 🛠️ Tecnologias

- Python / Flask
- Flask-SQLAlchemy / SQLite
- HTML5 / CSS3

## ⚙️ Como rodar

1. Clone o repositório.
2. Instale as dependências: `pip install -r requirements.txt`.
3. Execute: `python app.py`.
